# Proyecto Final: Aplicación Web de Análisis de Ventas de Vehículos

## Descripción General del Proyecto

Este repositorio contiene el código de una aplicación web interactiva desarrollada con **Streamlit** para realizar un analisis exploratorio de datos sobre anuncios de venta de coches en Estados Unidos.

El objetivo principal de este proyecto es demostrar habilidades de **Ingenieria de Software**, incluyendo:
1.  Gestión de control de versiones con **Git** y **GitHub**.
2.  Creación y gestión de **entornos virtuales** de Python.
3.  Desarrollo de una aplicación web interactiva con **Streamlit**.
4.  Despliegue continuo (CI/CD) usando **Render.com**.
